前言

這是我的第一篇部落格文章，之前沒有寫過部落格，想說今天就來試試。
Hexo安裝並沒有非常複雜，比想像中的簡單

這一行就可以做全局安裝，並沒有什麼需要額外配置的地方
部屬平台就選github吧，純前端也不需要什麼後台的功能，
github page就完事了
_config.yml就問google大神怎麼設定就好，一些常用的配置稍微記一下，
反正也很快就忘，忘了再找google大神就好

也不知道有什麼好寫的，總之今天一直在下雨，外面風很大，出去吃個飯還要撐傘。
整天下雨不知道在下三小，明明前幾天都還很熱，也不知道前幾天在熱三小= =

守望先鋒之亂

昨天蘇寧在送守望先鋒暢玩版，然後某寶各大店商也陸續加入

蘇寧網頁可以抽獎，但是我沒大陸手機號不能註冊帳號，mmp
後來看壇裡老哥說可以關注蘇寧微信公眾號參加，
